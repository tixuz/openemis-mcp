# Подсчет свободных должностей (на уровне учреждения или всей системы, с возможностью фильтрации по роли)

> 📖 **Read-only server.** Playbooks that create or update records require **[openemis-mcp-pro](https://github.com/tixuz/openemis-mcp-pro)**.

**Домен:** Персонал · **Аудитория:** admin, hr

Должность считается вакантной, если либо (а) ей не назначен ни один сотрудник, либо (б) общая эквивалентность полной занятости (FTE) всех сотрудников на этой должности составляет менее 1,00 (100%). Это охватывает как случаи «не назначен никто», так и случаи «недостаточный штат». Запрос может быть ограничен одним учреждением или выполнен по всей системе. Для фильтрации по конкретной роли — например, учителю — необходимо сопоставить данные с названиями должностей.

---

## Ресурсы

| Resource | Role in this workflow |
|---|---|
| `institution-positions` | Основной список определенных должностей — каждая строка является слотом, который может быть заполнен одним или несколькими сотрудниками. |
| `institution-staff-position-profiles` | Связывает сотрудников с их должностными обязанностями и используется здесь для проверки наличия активных занимающих эту должность лиц. |
| `staff-position-titles` | Каталог должностных названий (Учитель, Директор и т. д.) — используется для фильтрации должностей по роли, когда пользователь запрашивает определенный тип. |
| `institutions` | Предоставляет названия и коды учреждений при ограничении запроса одной школой или поиске учреждения по названию. |

## Шаги

| # | Title | Tool | Purpose |
|---|---|---|---|
| 1 | Разрешение идентификатора названия роли (если фильтруете по роли) | `openemis_get` | Получите id для названия должности, соответствующего запрошенной роли (например, "учитель"). |
| 2 | Получение должностей для данного учреждения | `openemis_get` | Перечислите все должности в целевом учреждении, опционально отфильтрованные по разрешенному идентификатору названия. |
| 3 | Получение должностей для всей системы (если учреждение не указано) | `openemis_get` | Получите постраничный список всех должностей по всей системе, опционально отфильтрованных по названию. |
| 4 | Получение сотрудников, назначенных на каждую должность | `openemis_get` | Для каждой должности получите строки сотрудников, чтобы проверить, назначен ли кто-нибудь и какова их совокупная FTE. |
| 5 | Подсчет свободных должностей | — | Должность считается вакантной, если нет активных строк сотрудников или если сумма их значений FTE ниже 1,0. |
| 6 | Возврат количества | — | Сообщите общее количество свободных должностей, с опциональной детальной информацией о недоукомплектованных местах и их текущем FTE. |

### Примечания к шагам

**Шаг 1 — Разрешение идентификатора названия роли:** Вызовите `staff-position-titles` с ограничением в 200 и запросите только поля `id` и `name`. Затем найдите записи, в названии которых содержится запрашиваемое слово роли (без учета регистра). Запишите соответствующий `id`, чтобы использовать его в качестве фильтра на шагах 2 или 3.

**Шаг 4 — Получение сотрудников, назначенных на каждую должность:** Для каждого id должности, полученного на шагах 2 или 3, вызовите `institution-staff`, отфильтрованный по `institution_position_id`. Запросите `id`, `FTE`, `staff_status_id` и `end_date`. FTE возвращается в виде строки, например `"1.00"` — проанализируйте ее как число с плавающей запятой перед суммированием. Учитывайте только те строки сотрудников, которые в настоящее время активны (не имеют даты окончания или не уволились).

**Шаг 5 — Подсчет свободных должностей:** Должность вакантна, когда (возвращенные строки = 0) ИЛИ (сумма `parseFloat(row.FTE)` по активным строкам < 1,0). Оба условия означают должность, которую необходимо заполнить.

---

## Примечания

- FTE в ответе API является строкой вида `'1.00'` — проанализируйте ее как число с плавающей запятой перед суммированием.
- Учитывайте только строки `institution_staff`, которые в настоящее время активны — обычно `staff_status_id` соответствует коду «Назначен» или «Текущий».
- НЕ включайте сотрудников с датой окончания или уволившихся в сумму FTE (они освобождают должность).
