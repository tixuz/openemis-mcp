# Обзор отчета по успеваемости на уровне класса за класс

> 📖 **Read-only server.** Playbooks that create or update records require **[openemis-mcp-pro](https://github.com/tixuz/openemis-mcp-pro)**.

**Домен:** Отчет · **Аудитория:** учитель, администратор

Получите конфигурацию годового отчета для класса, перечислите предметы, включенные в него, и извлеките результаты оценок по всем ученикам этой параллели и периода. Это дает общий классный обзор, эквивалентный тому, что учитель видит на экране сводного отчета по успеваемости. Обратите внимание, что это рабочий процесс только для чтения — генерация PDF требует отдельного асинхронного процесса (см. плейбук `generate-student-report-card-pdf`).

---

## Ресурсы

| Ресурс | Роль в этом рабочем процессе |
|---|---|
| `report-cards` | Определение годового отчета для учреждения и академического периода — определяет, какие предметы и компоненты оценивания включены в официальный отчет. |
| `institution-class-grades` | Предоставляет идентификатор(ы) класса по образовательной программе (`education_grade_id`), присвоенный этому классу, используется для фильтрации результатов оценки по параллели. |
| `report-card-subjects` | Перечисляет предметы в годовом отчете с их порядком отображения — обеспечивает соответствие обзора класса официальному макету годового отчета. |
| `assessment-item-results` | Предоставляет все строки оценок учеников для этой параллели и периода — источник данных для сводной таблицы на уровне класса. |

## Шаги

| # | Название | Инструмент | Цель |
|---|---|---|---|
| 1 | Получить конфигурацию годового отчета | `openemis_get` | Извлечь определение годового отчета, чтобы подтвердить, какие предметы и шкала оценивания входят в область действия. |
| 2 | Получить назначения параллели для класса | `openemis_get` | Извлечь идентификатор(ы) класс по образовательной программе (`education_grade_id`) для этого класса, чтобы ограничить запросы результатов оценки. |
| 3 | Получить предметы в годовом отчете | `openemis_get` | Перечислить предметы, включенные в годовой отчет, и их порядок отображения. |
| 4 | Получить результаты оценки для класса | `openemis_get` | Извлечь все строки оценок учеников для этой параллели и периода, чтобы составить сводную таблицу на уровне класса. |

### Примечания к шагам

**Шаг 1 — Получить конфигурацию годового отчета:** Отфильтруйте `report-cards` по `institution_id` и `academic_period_id`. Ответ включает `report_card_id` (необходим для шага 3), ссылку на шкалу оценивания и информацию о включении комментариев учителя. Если для одного периода существует несколько годовых отчетов, сопоставьте их по `education_grade_id` или спросите пользователя, какой использовать.

**Шаг 2 — Получить назначения параллели для класса:** Отфильтруйте `institution-class-grades` по `institution_class_id`. Обычно класс соответствует одной параллели, но некоторые учреждения настраивают классы с несколькими параллелями — сохраните все возвращенные значения `education_grade_id` для использования в шагах 3 и 4.

**Шаг 3 — Получить предметы в годовом отчете:** Отфильтруйте `report-card-subjects` по `report_card_id` из шага 1. Каждая строка связывает предмет с годовым отчетом через `education_subject_id`. Используйте этот список, чтобы определить, какие предметы следует отображать в обзоре класса — предметы, не включенные в этот список, не появляются в официальном годовом отчете.

**Шаг 4 — Получить результаты оценки для класса:** Отфильтруйте `assessment-item-results` по `education_grade_id` (из шага 2) и `academic_period_id`. Для большого класса используйте постраничный просмотр с `limit=200`. Каждая строка содержит `student_id`, `assessment_item_id`, `marks` и `grading_option_id`. Группируйте по `student_id` и `assessment_item_id`, чтобы построить таблицу для каждого ученика по каждому предмету. Строки с `marks=null` указывают на оценки, которые еще не введены.

## Пример запроса

> «Покажи мне обзор годового отчета для всех учеников 6 класса — как каждый ученик справляется по предметам?»

Агент будет:
1. Вызвать `openemis_discover` или `openemis_list_playbooks`, чтобы найти этот плейбук
2. Следовать приведенным выше шагам
3. Вернуть ответ на простом английском языке
