# संस्थान सांख्यिकी PDF रिपोर्ट जनरेट करें

> 📖 **Read-only server.** Playbooks that create or update records require **[openemis-mcp-pro](https://github.com/tixuz/openemis-mcp-pro)**.

**डोमेन:** रिपोर्ट · **दर्शक:** प्रशासक

प्रशासक एक संस्थान के लिए नामांकन, कर्मचारी और उपस्थिति सांख्यिकी को कवर करने वाली एक सारांश PDF का अनुरोध करता है। यह प्लेबुक डेटा लुकअप चरणों का वर्णन करती है; वास्तविक PDF OpenEMIS एप्लिकेशन के अंदर जनरेट की जाती है।

---

## संसाधन

| संसाधन | इस वर्कफ़्लो में भूमिका |
|---|---|
| `institutions` | संस्थान आईडी और प्रोफ़ाइल प्रदान करता है — सभी सांख्यिकी क्वेरीज़ के लिए एंकर। |
| `report-card-processes` | रिपोर्ट जनरेशन जॉब्स के लिए सिस्टम-स्तरीय प्रक्रिया रिकॉर्ड्स — सांख्यिकी जनरेशन जॉब की स्थिति को ट्रैक करने के लिए उपयोग किया जाता है यदि ट्रिगर किया गया हो। |
| `institution-report-card-processes` | संस्थान-स्कोप्ड प्रक्रिया रिकॉर्ड्स जो इस विशिष्ट संस्थान के लिए PDF जनरेशन स्थिति को ट्रैक करते हैं। |

---

## चरण

| # | शीर्षक | टूल | उद्देश्य |
|---|---|---|---|
| 1 | रिपोर्ट टेम्पलेट की पहचान करें | `openemis_get` | संस्थान सांख्यिकी के लिए कॉन्फ़िगर किए गए कस्टम रिपोर्ट टेम्पलेट को देखें। |
| 2 | जनरेशन शुरू करें | `openemis_create` | PDF जनरेशन शुरू करने के लिए टेम्पलेट आईडी और संस्थान आईडी के साथ report-card-processes पर POST करें। |
| 3 | पोल करें और डाउनलोड करें | `openemis_get` | स्थिति "completed" तक पहुँचने तक प्रक्रिया स्थिति की समय-समय पर जाँच करें, फिर डाउनलोड लिंक प्राप्त करें। |

### चरण नोट्स

**चरण 1 — रिपोर्ट टेम्पलेट की पहचान करें:** ध्यान दें कि `custom-reports` v5 REST API के माध्यम से उपलब्ध नहीं है। सांख्यिकी PDF जनरेशन OpenEMIS एप्लिकेशन के अंदर रिपोर्ट इंजन का उपयोग करती है। इस चरण में इस संस्थान पर कॉन्फ़िगर किए गए उपलब्ध टेम्पलेट्स के लिए `report-card-processes` या `institution-report-card-processes` की जाँच शामिल होनी चाहिए।

**चरण 2 — जनरेशन शुरू करें:** यदि टेम्पलेट v5 API के माध्यम से एक्सेसिबल है, तो टेम्पलेट आईडी, संस्थान आईडी और किसी भी डेट रेंज पैरामीटर्स के साथ `report-card-processes` पर POST करें। यदि टेम्पलेट v5 के माध्यम से एक्सेसिबल नहीं है, तो उपयोगकर्ता को एप्लिकेशन UI से PDF ट्रिगर करने के लिए निर्देशित करें।

**चरण 3 — पोल करें और डाउनलोड करें:** एक बार प्रक्रिया आईडी प्राप्त होने के बाद, इसकी स्थिति जाँचने के लिए प्रक्रिया आईडी के साथ `institution-report-card-processes` को कॉल करें। जब स्थिति "completed" तक पहुँच जाए, तो प्रतिक्रिया में जनरेट की गई PDF के लिए एक URL या फ़ाइल पथ शामिल होगा।

> "लिंकन प्राइमरी स्कूल के लिए वार्षिक सांख्यिकी PDF जनरेट करें।"

एजेंट यह करेगा:
1. इस प्लेबुक को खोजने के लिए `openemis_discover` या `openemis_list_playbooks` को कॉल करें
2. उपरोक्त चरणों का पालन करें
3. उत्तर सरल भाषा में वापस करें

## नोट्स

⚠️ **नोट:** `custom-reports` v5 REST API के माध्यम से उपलब्ध नहीं है। सांख्यिकी PDF जनरेशन एप्लिकेशन के अंदर रिपोर्ट इंजन का उपयोग करती है। यह प्लेबुक केवल डेटा लुकअप को कवर करती है — PDF को एप्लिकेशन UI से ट्रिगर करें।
